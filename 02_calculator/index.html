Calculator

you will build a simple calculator script with the following methods:

add takes two parameters and adds them

subtract takes two parameters and subtracts the second from the first

sum takes an array of parameters and adds them all together

Hint:

There are several ways to complete sum. You may use loops (e.g. while) to get the tests to pass. You could also use the reduce function from the functools package.

Learning to write tests:

There are also some exercises that will help teach you test-driven development, not simply test-guided learning.

Your mission is to write tests for three new functions (and then implement the functions so that the tests pass):

  • multiply which multiplies two numbers together
  • power which raises one number to the power of another number
  • [factorial](http://en.wikipedia.org/wiki/Factorial) (check Wikipedia if you forgot your high school math).
