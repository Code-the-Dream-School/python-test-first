hello

Hello!

This lab teaches basic Python function syntax.

Open a terminal in this directory

cd 00_hello

This directory is the starting point for this exercise. It contains two Python files, one being the test, and one being the Python file to edit to make the specs pass.

Run the test

pytest -x test.py

Watch it fail

you should see an error like this:

AttributeError: module 'hello' has no attribute 'hello'

Create the hello function

Fix this by opening hello.py and creating an empty function:

def hello:
    pass 
    # The pass statement gives the function 
    # a body, but doesn't do anything

Save it. Run the test again.

Watch it fail

Now you should see an error like this:

AssertionError: assert None == 'Hello!'

This means that while it found the file, and it found the function, it's not returning anything! ("None" is the Python way of saying "not anything".)

Make it return something

Inside the "hello" function, put a single line returning a string that is not "Hello!". (Here we are simulating you making an honest mistake, so we can see what the error message looks like.)

def hello:
    return "whuh?"

Save it. Run the test again.

Watch it fail

Now you should see an error like this:

AssertionError: assert 'whuh?' == 'Hello!'

Correct this by changing "whuh?" to "Hello!". Save it. Run the test again.

Watch it pass! Well, not quite.

Anyway, the first test passed. You'll see status at the bottom that says:

1 failed, 1 passed

Give yourself a high five

Also, sing a song and do a little dance.

And then...

Fix the next failure! :-)

You are running pytest with the -x option. When it is run this way, it stops at the first failure. You could try it without the -x if you want to see how many tests you need to fix.

AttributeError: module 'hello' has no attribute 'greet'

In order to get the next test to pass, you'll need to add a greet function that accepts an argument.

def greet(who):
    return f"Hello, {who}!"
